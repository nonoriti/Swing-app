Como jogar

O campo de Golfe

Um campo de golfe oficial é composto por 18 buracos, podendo também existir campos com 9 buracos, onde os jogadores realizam duas voltas para completar o percurso. Cada buraco é dividido em várias zonas, cada uma com características específicas que influenciam a estratégia e a execução das tacadas.

Locais do Campo

Tee (Área de Partida)

O tee é a área onde o jogador inicia cada buraco. Normalmente, existem diferentes tees, com distâncias variadas, para acomodar jogadores de diferentes níveis de habilidade. Esta é uma área plana e relvada, onde o jogador posiciona a bola para a primeira tacada.

Fairway

O fairway é a zona principal do percurso, com relva cortada a uma altura baixa, facilitando o jogo. É o caminho ideal para as tacadas intermediárias, levando a bola em direção ao green. A largura do fairway pode variar, mas geralmente é entre 25 a 30 metros.

Rough

O rough é a área que circunda o fairway, com relva mais alta e densa, dificultando o controle da bola. É considerado uma zona de penalidade, onde o jogador pode enfrentar desafios adicionais.

Bunkers

Os bunkers são obstáculos de areia estrategicamente posicionados ao longo do percurso, tanto no fairway quanto ao redor do green. Eles adicionam dificuldade ao jogo, exigindo habilidades específicas para superar.

Green

O green é a área final onde se encontra o buraco com a bandeira. A relva é cortada muito rente, permitindo que a bola role suavemente. É o local onde o jogador realiza o putt para completar o buraco.
